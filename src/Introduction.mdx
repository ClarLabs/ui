## Usage

```tsx

import '@clarlabs/ui/dist/index.css';

function App() {
  return (
    <Card title="Welcome">
      <Input placeholder="Enter your name" />
      <Button variant="primary">Submit</Button>
    </Card>
  );
}
```

## Theme Customization

Toggle between light and dark themes by adding the `light-mode` class to your root element:

```tsx
// Toggle theme
document.body.classList.toggle('light-mode');
```
